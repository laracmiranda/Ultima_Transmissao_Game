## 🎮 Como Jogar

### Controles

| Tecla     | Ação             |
| --------- | ---------------- |
| `W A S D` | Movimentação     |
| `E`       | Interagir        |
| `Enter`   | Avançar diálogos |

---

## ✨ Funcionalidades

* Gameplay narrativa com múltiplas tentativas
* Eventos especiais em cada escolha incorreta
* Sistema de diálogos com efeito Typewriter
* Cutscenes animadas em pixel art
* Trilha sonora dinâmica
* Sistema de Game Over contextual
* Créditos automáticos
* Arte inspirada em jogos clássicos de Game Boy

## 🧠 Sobre o Projeto

A última transmissão foi desenvolvido utilizando Unity e C# com foco em:

* Narrativa interativa
* Sistemas de UI
* Controle de estados de jogo
* Animações frame-by-frame
* Organização de código para projetos 2D

O projeto foi criado como um presente de dia dos namorados, para poder trazer uma experiência única e divertida!
